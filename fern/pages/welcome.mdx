## What is this?

The AR Glasses Master SDK is a comprehensive documentation project that extracts, documents, and unifies the APIs from **27+ open-source repositories** across the AR/XR glasses ecosystem.

## Covered Devices

| Device | Vendor | SDK Type | Key Capabilities |
|--------|--------|----------|-----------------|
| XREAL Air/Air 2/Air 2 Ultra | XREAL | Unity XR Plugin | 3DOF/6DOF, Camera, Spatial Anchors |
| RayNeo X2/X3 Pro | RayNeo | Android ARDK + Unity | 3DOF/6DOF SLAM, Gaze, Face Detection |
| Vuzix Blade 2 | Vuzix | Android HUD SDK | ActionMenu UI, Widget, Notifications |
| Everysight Maverick | Everysight | iOS/Android Binary SDK | HUD, BLE, Sensors, OTA |

## API Domains

The SDK covers **12 functional domains** with **489 documented endpoints**:

1. **Display and Rendering** — Binocular, WebXR, Stereo, HUD, Curved
2. **IMU and Head Tracking** — Fusion AHRS, Complementary Filter, Quaternion Math
3. **Camera and Computer Vision** — SLAM, Face Detection, Image Recognition
4. **Audio and Speech** — TTS, Mic Capture, Translation, Voice Assistant
5. **BLE and Wireless** — GATT NUS, PPoG, Wi-Fi P2P, WebSocket
6. **Gesture and Input** — Temple Touch, Gaze Ray, Hand Tracking, Ring
7. **Spatial Computing** — OpenXR, Monado, Plane Detection, Spatial Anchors
8. **ML/AI On-Device** — TFLite Micro, Groq LLM, SAM3
9. **GPS and Geolocation** — Phone GPS IPC, Location Streaming
10. **Device Management** — USB VID/PID, Hotplug, Firmware, Config
11. **3D Environment** — StardustXR Scenegraph, FlatBuffers, Wayland
12. **Geo/Maps Intelligence** — Overpass API, Gemini Maps Grounding
